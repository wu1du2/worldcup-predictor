# main_strategy

主策略：低比分篮子 v0。

规则：

1. 每场只买 `0-0`、`0-1`、`1-0`、`1-1`。
2. 每个比分 `1` 注。
3. 如果某个比分在本场没有赔率，则跳过该比分，不用其他比分补位。
4. 只使用 `context.json` 中的赛前信息，包括赛程、赛前赔率、赔率变化、赛前媒体信息、赛前伤病信息、赛前机构分析。
5. 禁止使用赛果、赛后报道、赛后评分、赛后技术统计、赛后赔率或任何发布时间晚于开球时间的信息。

输出格式：

```json
{
  "reason": "解释为什么该策略在这场比赛输出这些下注。",
  "stakes": [
    { "score": "0-0", "stake": 1 },
    { "score": "0-1", "stake": 1 },
    { "score": "1-0", "stake": 1 },
    { "score": "1-1", "stake": 1 }
  ]
}
```
